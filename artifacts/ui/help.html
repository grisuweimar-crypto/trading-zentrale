version 0.0.1-v7 - build 2026-02-13 - zurueck zum Dashboard

Privates Research-Dashboard. Keine Anlageberatung. Inhalte koennen unvollstaendig oder fehlerhaft sein.

60 Sekunden: So nutzt du das Dashboard

  1. Preset waehlen (z.B. ALL, CORE, TOP).
  2. Suche und Quick-Filter setzen.
  3. Bucket-Matrix anklicken fuer Fokus.
  4. Drawer oeffnen fuer Detailpruefung.
Glossar (kurz)

Score: internes Ranking. Confidence: Qualitaet/Konfluenz. R0-R5: Workflow-Code, kein Kaufsignal.

trend_ok / liquidity_ok: Mindestfilter. score_status: OK/AVOID/NA.

Bucket-Matrix

X-Achse = Score, Y-Achse = Risk. Klick auf ein Feld aktiviert einen zusaetzlichen Filter.

Market Context

Nur Kontext, kein Einfluss auf Scoring.

  • Breadth: Anzahl Gewinner/Verlierer im aktuellen gefilterten Universe.
  • Movers: staerkste Auf- und Abbewegungen.
  • Heatmap: Verteilung nach Cluster/Saeule und Score-Buckets.
  • Preset-Qualitaet: Median/IQR fuer Score/Confidence plus Trend/Liq-Anteile.
Pipeline (Future-Me)

python -m scanner.app.run_daily erstellt Watchlist/Reports.
python -m scanner.ui.generator erzeugt artifacts/ui/index.html.

Troubleshooting
  • Wenn UI leer ist: zuerst run_daily, dann ui.generator.
  • Wenn Texte komisch aussehen: UI neu generieren und Browser-Cache leeren.
  • Wenn Contract-Error: python scripts/validate_contract.py --csv artifacts/watchlist/watchlist_ALL.csv.
